# BTU 3-way checkback

對於所有 XYZ 叫序 (1X-1Y-1Z 或 1X-1Y-1NT)，本制度採用 BTU 風格的 checkback，總共
3 個人為叫品：

- **2♣** 含有多種可能
  - 想束叫 2 線花色
  - 特殊牌型的強力邀請
  - 平均牌邀請
  - 可再約定一種特殊的迫叫成局，準備再叫 3NT
- **2♦** 迫叫成局
- **2NT** 轉換至梅花

由此可見，直接叫 2 線花色變成大範圍邀請，而 3 線花色則為滿貫試探。和公版 XYZ
相比，BTU 版把直接叫 2 線花色和透過 2♣ 再叫的意義對調了，因為邀請更怕對手干擾。
